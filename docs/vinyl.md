### 🎵 Step 4_2: Vinyl Station Interaction & Audio Workflow (ระบบเครื่องเล่นแผ่นเสียงและการจัดการระบบเสียง)

1. State & Logic Flow (ระบบจัดการสถานะและตรรกะตัวแปร)
- เครื่องมือที่ใช้: React State (useState) และ React Context API (สำหรับคุมเสียง Global)
- ตรรกะการทำงาน:
  * ใช้ State บันทึกแผ่นเสียงที่ถูกเลือก (เช่น activeVinylId: number | null) เพื่อเก็บค่าตัวเลขแผ่นเสียงลำดับที่ 1 ถึง 12 ที่กำลังใช้งานอยู่บนเครื่องเล่น หากเป็น null หมายถึงยังไม่มีการเลือกแผ่น
  * ใช้ State ควบคุมสถานะการเล่นเสียง (เช่น isPlaying: boolean) เพื่อตรวจจับว่าสถานะเพลงกำลังเล่นอยู่หรือหยุดเล่น เพื่อส่งสัญญาณไปสั่งการแอนิเมชันการหมุนของตัวแผ่น และตำแหน่งองศาของก้านหัวเข็ม (Tone Arm) ว่าต้องยกขึ้น วางลง หรือดีดกลับแท่นพัก
  * มีระบบตรวจจับสถานะพิกัดความใกล้ (Drop Zone Proximity) เพื่อเปิดใช้ตัวแปรสถานะ Boolean คอยเปิด-ปิดไฮไลต์พื้นที่รับแผ่นเสียงเมื่อแฟนลากแผ่นเข้าใกล้รัศมีตัวเครื่องเล่น

2. Asset & UI Layout Design (การจัดวางโครงสร้างเลเยอร์หน้าจอและภาพหน้าปก)
- เครื่องมือที่ใช้: Tailwind CSS 100%
- ตรรกะการทำงาน:
  * โครงสร้างกล่องเก็บแผ่นเสียง (Vinyl Storage Box): ออกแบบกล่องแนวตั้งสำหรับจัดเก็บแผ่นเสียงทั้ง 12 แผ่นให้วางเรียงซ้อนกันอย่างมีระเบียบ จัดวางในตำแหน่งที่สอดรับกับขนาดหน้าจอมือถืออัตโนมัติ
  * ระบบล็อกสัดส่วนรูปภาพกึ่งกลางแผ่น (UI Center Scaling): จัดวางบล็อกรูปคู่หน้าปก (`cover-01.png` ถึง `cover-12.png`) ไว้กึ่งกลางแผ่นเสียงสีดำเป๊ะ ๆ ด้วยคำสั่งจัดพิกัดกึ่งกลางสัมบูรณ์ (Absolute Centering: top-1/2 left-1/2 transform -translate-x-1/2 -translate-y-1/2)
  * **เทคนิคล็อกสัดส่วนภาพไม่ให้บิดเบี้ยว:** บังคับไฟล์รูปภาพหน้าปกทุกใบด้วยคลาส `object-cover` และ `rounded-full` เสมอ เพื่อสั่งให้เบราว์เซอร์บีบและตัดขอบรูปภาพดุมกลางให้เป็นวงกลม 100% โดยอัตโนมัติ โดยที่สัดส่วนภาพเดิม (Aspect Ratio) จะไม่เกิดอาการบี้แบน ยืด หรือขยายจนเพี้ยน ไม่ว่ารูปต้นฉบับที่ส่งเข้ามาจะเป็นทรงสี่เหลี่ยมจัตุรัส แนวตั้ง หรือแนวนอนก็ตาม
  * แผงปุ่มควบคุมพาสเทล: จัดวางแผงปุ่มกดสไตล์มินิมอล 3 ปุ่ม (Play, Pause, Stop) ไว้ที่ส่วนฐานล่างของตัวเครื่องเล่นแผ่นเสียงเพื่อให้ง่ายต่อการกดใช้งาน

3. Drag & Drop / Click Swap Interaction (ระบบลากวางและกลไกสลับแผ่นอัตโนมัติ)
- เครื่องมือที่ใช้: Framer Motion (คำสั่ง drag และ dragTransition)
- ตรรกะการทำงาน:
  * ระบบลากวางอิสระ (Drag & Drop): เปิดฟังก์ชันคำสั่งลากคลุมบนตัวแผ่นเสียงทั้ง 12 ใบ แฟนสามารถใช้นิ้วลากแผ่นขยับย้ายตำแหน่งบนจอได้อย่างอิสระ หากปล่อยนิ้วนอกขอบเขตเครื่องเล่น ตัวแผ่นจะเล่นอนิเมชันเด้งกลับสล็อตกล่องเก็บตัวเองทันที
  * ตรรกะการสลับแผ่นอัตโนมัติ (Auto-Swap): หากมีแผ่นเก่า (เช่น แผ่นที่ 1) คาอยู่บนเครื่องเล่น แล้วแฟนลากแผ่นใหม่ (เช่น แผ่นที่ 2) ลงมาปล่อยทับพื้นที่เป้าหมาย ระบบจะสั่งการเปลี่ยนผ่านทันที โดยแผ่นเก่าจะรันอนิเมชันลอยดึ๋งเด้งกลับเข้าที่เก็บในกล่องของตัวเองโดยอัตโนมัติ และสั่งดูดแผ่นใหม่เข้าล็อกกึ่งกลางเครื่องเล่นแทน
  * ระบบจิ้มสลับทางเลือก (Click-to-Swap): ทำฟังก์ชันรองรับในกรณีที่แฟนไม่ได้ลากแผ่นเสียง แต่ใช้วิธีกดจิ้ม (onClick) ที่แผ่นเสียงในกล่องเก็บโดยตรง ตัวแผ่นเสียงใบนั้นจะเล่นอนิเมชันสไลด์ลอยลอยข้ามหน้าจอจากพิกัดกล่องมาวางลงล็อกตรงกลางเครื่องเล่นให้เองโดยอัตโนมัติ

4. Global Audio Playback Sequence (ระบบจัดการเสียงแบ็กกราวด์และลำดับการเล่น)
- เครื่องมือที่ใช้: Web Audio API หรือไลบรารีระบบจัดการเสียงที่ฝังที่รากบนสุด (Root Component)
- ตรรกะการทำงาน:
  * ระบบเสียงไร้รอยต่อ (Continuous Background Music): ตัวเล่นไฟล์เสียงจะถูกจัดวางไว้บนโครงสร้างระดับสูงสุดของเว็บ เพื่อให้เพลงสามารถสตรีมมิ่งคลอเป็นพื้นหลัง (BGM) ยาวไปได้เรื่อย ๆ อย่างต่อเนื่อง ไม่มีอาการเสียงสะดุดหรือเริ่มเล่นใหม่ แม้ว่าแฟนจะกดปิดหน้าต่างเครื่องเล่นเสียงนี้เพื่อไปเปิดอ่านไดอารี่หรือแวะไปเล่นมินิเกมตอติญ่าชิ้นอื่นก็ตาม
  * ลำดับขั้นตอนการสลับเพลง (Playback Sequence Flow): 
    - ทันทีที่มีการเปลี่ยนแผ่นเสียงสำเร็จ ➡️ ระบบสั่งคำสั่งหยุดเล่นไฟล์เพลงเก่าทันที ➡️ สั่งโหลดไฟล์เสียงเพลงใหม่มารองรับในบัฟเฟอร์ ➡️ สั่งก้านหัวเข็ม (Tone Arm) เล่นอนิเมชันหมุนเลื่อนมาวางแปะลงบนขอบนอกของแผ่นเสียง ➡️ ดีเลย์จังหวะเวลาหน่วงเล็กน้อยเพื่อให้สมจริง ➡️ สั่งเริ่มเล่นเสียงเพลงใหม่ พร้อมเปิดสวิตช์ส่งสัญญาณให้แผ่นไวนิลสีดำและรูปคู่ดุมกลางเริ่มหมุนติ้วๆ รอบตัวเองแบบต่อเนื่องวนลูป

5. Control Button Workflow (กระบวนการทำงานของปุ่มควบคุมทั้ง 3 ปุ่ม)
- เครื่องมือที่ใช้: React Logic ร่วมกับ Framer Motion Animation
- ตรรกะการทำงาน:
  * ปุ่ม Play (เล่นเพลง):
    - จะอนุญาตให้สั่งงานได้ก็ต่อเมื่อตรวจเช็กพบสถานะ `activeVinylId !== null` (มีแผ่นวางอยู่บนเครื่อง) เท่านั้น
    - เมื่อกดใช้งาน: ระบบจะสั่งเล่นเสียงเพลงต่อจากวินาทีเดิมที่ค้างไว้, สั่งให้ก้านหัวเข็มเลื่อนมาวางบนหน้าแผ่นเสียง, และสั่งเปิดแอนิเมชันแผ่นไวนิลรวมถึงรูปปกคู่กลางแผ่นให้เริ่มหมุนรอบตัวเองทันที
  * ปุ่ม Pause (หยุดชั่วคราว):
    - เมื่อกดใช้งาน: ไฟล์เพลงจะหยุดนิ่งค้างอยู่ ณ วินาทีนั้น ๆ ทันที, แผ่นไวนิลและภาพคู่ตรงกลางจะหยุดหมุนนิ่งสนิททันทีในองศาเดิม, แต่ตรรกะก้านหัวเข็มจะยังคงวางคาอยู่บนแผ่นเสียงเหมือนเดิม (ไม่ดีดตัวกลับ) เพื่อเตรียมพร้อมสลับกลับมาเล่นเพลงต่อได้ทันที
  * ปุ่ม Stop (หยุดสนิท):
    - เมื่อกดใช้งาน: ระบบจะสั่งหยุดการทำงานของเสียงและรีเซ็ตค่าตัวนับเวลาของเพลงกลับไปเริ่มต้นที่นาที 0:00 ใหม่ทั้งหมด, สั่งแผ่นเสียงและรูปหน้าปกให้หยุดหมุน, และสั่งก้านหัวเข็มเล่นอนิเมชันยกตัวลอยขึ้นแนวดิ่งแล้วเลื่อนหมุนกลับไปเก็บ ณ แท่นวางตั้งต้นของตัวเองโดยอัตโนมัติ
